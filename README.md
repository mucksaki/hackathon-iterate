# Hackathon Iterate - Clara

An intelligent voice notes assistant called Clara that lets you record audio notes and interact with them through natural conversation. Ask questions about your recorded notes and receive spoken answers powered by AI transcription and speech synthesis.

## âœ¨ Features

- **Voice Recording** - Capture audio notes directly in your browser
- **Smart Transcription** - Automatic speech-to-text conversion with speaker identification
- **Conversational Q&A** - Ask questions about your notes using natural language
- **Voice Responses** - Get answers spoken back to you in speech mode
- **Note Management** - Organize and search through your voice note library

# api keys
add your `GRADIUM_API_KEY`, `PYANNOTE_API_KEY` and `GOOGLE_API_KEY` keys to an `.env` file in the root.

## Commands

- `make back-test` - Start the backend
- `make front-test` - Start the frontend
